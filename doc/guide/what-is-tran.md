# VTran 是什么？

Tran是一个轻量级的局域网文件传输工具及互相聊天工具，在内网开发中，可能受环境局限，无法使用未信任的软件，Tran可以解决这个问题，
因为它基于``浏览器``，不需要安装额外的软件，只需要前端人员拉下代码直接部署即可

## 使用场景

- 局域网内文件传输

 利用``Node.js`` 进行文件上传操作，使用`websocket` 进行推送到用户消息

- 聊天

 当用户注册成功后即可使用此功能进行局域网内部成员聊天
